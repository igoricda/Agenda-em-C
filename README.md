Implementação de agenda em C, usando arquivos e uma árvore binária como indice de memória.
Tem como funcoes: [1] Inserir registro
                  [2] Listar registros
                  [3] Buscar registro
                  [4] Alterar registro
                  [5] Apagar Registro
                  [6] Recuperar Registro
                  [7] Listar lixeira
                  [8] Esvaziar lixeira
